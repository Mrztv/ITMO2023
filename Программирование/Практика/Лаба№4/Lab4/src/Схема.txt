локальных, анонимных и вложенных (static non-static)
Локальные - можно поместить где угодно, наверное лучше поместить внутрь какого-то метода!!
анонимные - чаще всего используется при передаче как параметр в метод, вместо него может быть использован локальный класс, если требуется неоднократно использовать класс
внутренний класс(non-static) - класс вложеный в другой класс, не существует объект без объекта внешнего класса
внутренний класс(static) - работает на подобии с  статик полями и методами, может быть вызван без создания объекта внешнего класса
внутренние классы используются для тех моментов когда нужен специфичный тип данных который используется только внутри класса
